Getting Started

Table of Contents

Node Version Mananger ​

  1. if you are a windows user, recommended use nvm-windows
  2. if you are a Mac user, recommended use n or fnm

Node.js and Vue.js Requirements ​

  • vue>=3.2.0，node>=18.10.0

Install ​

sh
npm install @cztrina/design-vue
sh
yarn add @cztrina/design-vue
sh
pnpm add @cztrina/design-vue

Full Import ​

js
import { createApp } from 'vue'
import App from './App.vue'
import TnDesV from '@cztrina/design-vue'
import '@cztrina/design-vue/es/theme-chalk/index.css'

const app = createApp(App)
app.use(TnDesV)
app.mount('#app')

On-Demand Import ​

  1. Install unplugin-vue-components
sh
npm i unplugin-vue-components -D
sh
yarn add unplugin-vue-components -D
sh
pnpm add unplugin-vue-components -D
  1. Configure vite.config.js
vite.config.js
js
import { defineConfig } from 'vite'
import Components from 'unplugin-vue-components/vite'
import { tnsResolver } from '@cztrina/design-vue/lib/util/tnsResolver'

export default defineConfig({
  plugins: [
    Components({
      resolvers: [
        tnsResolver()
      ],
    }),
  ],
})
